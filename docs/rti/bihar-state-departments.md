# Drafted RTI applications — Bihar state departments

Ready-to-file applications covering the catalogue's no-metric schemes, grouped by the department
that administers them (RTIs are filed per-PIO, so one application covers several schemes). Fill the
applicant block from the template in `README.md`, adjust the FY range to the latest available, and
file. The five data points map directly to the `scheme_metrics` dimensions
(beneficiaries / budget / district / demographics / outcomes).

> **Period for all applications:** Financial Years **2021-22 to 2024-25** (latest available).
> Request the information **in electronic form (email/CD)** under Section 7(9).

---

## 1. Department of Agriculture, Government of Bihar
**Schemes:** Bihar Diesel Anudan Yojana · Bihar Krishi Yantra Anudan Yojana · Bihar Mukhyamantri
Bagwani Mission · Bihar Mukhyamantri Niji Nalkoop Yojana · Jananayak Karpoori Thakur Kisan Samman Nidhi.

For **each** scheme above, please provide:
1. **Year-wise number of beneficiaries** (farmers/applicants who received the benefit), FY 2021-22 to 2024-25.
2. **Year-wise funds** — amount **allocated** (budget estimate) and amount **actually disbursed** (₹), each FY.
3. **District-wise number of beneficiaries** across all districts of Bihar, for the latest completed FY.
4. **Category-wise split** of beneficiaries — by social category (SC / ST / EBC / BC / General) and by
   land-holding class (raiyat / non-raiyat) where recorded, latest FY.
5. **Applications received, sanctioned, and rejected** (with main reasons for rejection), each FY.

---

## 2. Education Department, Government of Bihar
**Schemes:** Mukhyamantri Balak/Balika Cycle Yojana · Mukhyamantri Balika Protsahan Yojana (Matric) ·
Mukhyamantri Kanya Utthan Yojana.

For **each** scheme above, please provide:
1. **Year-wise number of beneficiary students** (and amount paid per beneficiary), FY 2021-22 to 2024-25.
2. **Year-wise funds** allocated and disbursed (₹), each FY.
3. **District-wise number of beneficiary students**, latest completed FY.
4. **Split of beneficiaries by gender and by social category** (SC / ST / EBC / BC / General / Minority), latest FY.
5. **Number of eligible applications received vs. paid**, and number of payments **failed/returned** (e.g.
   Aadhaar/bank mismatch), each FY.

## 3. Social Welfare Department, Government of Bihar
**Schemes:** Bihar e-Kalyan Scholarship · Bihar Scholarship Portal (Pre/Post-Matric) · Mukhyamantri
Gram Parivahan Yojana · Bihar Nishulk Coaching Yojana · Bihar Parivarik Labh Yojana · Bihar Parvarish Yojana.

For **each** scheme above, please provide:
1. **Year-wise number of beneficiaries**, FY 2021-22 to 2024-25.
2. **Year-wise funds** allocated and disbursed (₹), each FY.
3. **District-wise number of beneficiaries**, latest completed FY.
4. **Split of beneficiaries by gender and by social category** (SC / ST / EBC / BC / General / Minority), latest FY.
5. Scheme-specific outcome: for scholarships — applications received / sanctioned / rejected; for Gram
   Parivahan — vehicles purchased; for Parvarish — children currently supported, each FY.

## 4. Labour Resources Department, Government of Bihar (incl. Bihar Skill Development Mission)
**Schemes:** Bihar Pravasi Majdoor Durghatna Bima Yojana · Mukhyamantri Pratigya Yojana (Internship).

For **each** scheme above, please provide:
1. **Year-wise number of beneficiaries** (Pravasi Majdoor: claims received/settled; Pratigya: interns enrolled), FY 2021-22 to 2024-25.
2. **Year-wise funds** allocated and disbursed (₹), each FY.
3. **District-wise number of beneficiaries**, latest completed FY.
4. **Split by gender and social category**, latest FY.
5. Outcome: Pravasi Majdoor — claims **received vs. settled vs. rejected** + average settlement time;
   Pratigya — interns who **completed** the internship and the stipend disbursed, each FY.

## Central schemes — mine public data FIRST (RTI only if unpublished)
PMJJBY · PMSBY · PMKVY · Sukanya Samriddhi Yojana · Stand-Up India are central. Their headline
enrolment/disbursement figures are usually **already public** — check before filing:
- **PMJJBY / PMSBY** — Ministry of Finance / PIB annual enrolment + claims data (Jan Suraksha portal).
- **PMKVY** — MSDE / NSDC dashboards + annual reports (candidates trained / certified / placed).
- **Sukanya Samriddhi** — Ministry of Finance / NSI account + deposit statistics.
- **Stand-Up India** — the Stand-Up India portal dashboard (loans sanctioned to SC/ST/women, by state).

For these, file an RTI only for the **Bihar-state breakdown** if the public source is national-only.
